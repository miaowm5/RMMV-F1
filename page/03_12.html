方法

addChild ( child ) Object

向容器中添加子元素。

参数：

  • child Object

    待添加的子元素

返回值：

Object:
被添加的子元素

addChildAt ( child  index ) Object

在容器指定索引的位置添加子元素。

参数：

  • child Object

    待添加的子元素

  • index Number

    指定的索引

返回值：

Object:
被添加的子元素

isReady () Boolean

检测图块组是否可以开始渲染。

返回值：

Boolean:
若可以开始渲染，返回 true 

refresh ()

强制重绘整个 tilemap（元件地图）。

removeChild ( child ) Object

从容器中移除子元素。

参数：

  • child Object

    待移除的子元素

返回值：

Object:
被移除的子元素

removeChildAt ( index ) Object

移除容器指定索引位置的子元素。

参数：

  • index Number

    指定的索引

返回值：

Object:
被移除的子元素

setData ( width  height  data )

设置 tilemap（元件地图） 的数据。

参数：

  • width Number

    地图的横向元件数

  • height Number

    地图的纵向元件数

  • data Array

    地图数据的一维数组

update ()

在每一帧中刷新 tilemap（元件地图）。

属性

animationCount Number

自动元件的动画计数器。

bitmaps Array

地图元件的 Bitmap（位图） 对象数组。

children Array

[只读] tilemap（元件地图） 的子元素数组。

flags Array

图块组的标志列表。

height Number

游戏画面的高度（像素）。

horizontalWrap Boolean

tilemap（元件地图） 是否横向循环。

origin Point

tilemap（元件地图） 卷动的起始坐标。

parent Object

[只读]包含此 tilemap（元件地图） 的父容器。

tileHeight Number

单个元件的高度（像素）。

tileWidth Number

单个元件的宽度（像素）。

verticalWrap Boolean

tilemap（元件地图） 是否纵向循环。

width Number

游戏画面的宽度。
