方法

addLoadListener ( listner )

Add a callback function that will be called when the audio data is loaded.

参数：

  • listner Function

    回调函数

addStopListener ( listner )

Add a callback function that will be called when the playback is stopped.

参数：

  • listner Function

    回调函数

canPlayM4a () Boolean [static]

检测浏览器能否播放 m4a 文件。

Returns:

Boolean:
若浏览器支持播放 m4a 文件，返回 True 

canPlayOgg () Boolean [static]

检测浏览器能否播放 ogg 文件。

Returns:

Boolean:
若浏览器支持播放 ogg 文件，返回 True 

clear ()

清除音频数据。

fadeIn ( duration )

淡入音频。

参数：

  • duration Number

    淡入的时间（秒）

fadeOut ( duration )

淡出音频

参数：

  • duration Number

    淡出的时间（秒）

initialize ( noAudio ) Boolean [static]

初始化音频系统。

参数：

  • noAudio Boolean

    无声模式的标志

返回值：

Boolean:
若音频系统能够正常使用，返回 True

isError () Boolean

检测加载时是否有错误发生。

Returns:

Boolean:
若发生错误，返回 True 

isPlaying () Boolean

检测音频是否正在播放。

返回值：

Boolean:
若正在播放，返回 True 

isReady () Boolean

检测音频是否已经可以开始播放。

返回值：

Boolean:
若音频可以开始播放，返回 True 

play ( loop , offset )

播放音频。

参数：

  • loop Boolean

    是否要循环播放

  • offset Number

    播放开始的时间（秒）

seek ()

获取当前音频的播放位置。

stop ()

停止播放音频。

属性

pan Number

音频的声场位置。

pitch Number

音频的音调。

url String

[只读] 音频文件的地址。

volume Number

音频的音量。
